Newsletter abonnieren

Wöchentlich oder monatlich – direkt in Ihr Postfach

⚙️ Einrichtungsanleitung (einmalig):
  1. Gehe zu app.brevo.com
  2. Navigiere zu Contacts → Forms
  3. Klicke Create a subscription form
  4. Füge Felder hinzu: Name, E-Mail, und zwei Checkboxen (Wöchentlich / Monatlich)
  5. Unter Settings → Share kopiere den Embed Code
  6. Ersetze den Platzhalter unten durch diesen Code und entferne diese Anleitung
